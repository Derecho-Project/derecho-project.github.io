static auto Foo::register_functions ( )
inlinestatic

All replicated objects must provide this static method, which should return a tuple containing all the methods that can be invoked by RPC.

Each method should be "tagged" with derecho::rpc::tag(), whose template parameter indicates which numeric constant will identify the method.

Returns
A tuple of "tagged" function pointers.
