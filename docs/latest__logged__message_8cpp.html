Detailed Description

Prints out the message number of the last message in a Derecho logfile.

Since "sequence numbers" depend on the number of members in a group, the least confusing way to print the message number is the tuple (view_id, sender, index). This is printed without the parentheses or comma for ease of parsing by bash scripts.
