◆ is_named

template<typename Row, typename ExtensionList>
using sst::PredicateBuilder< Row, ExtensionList >::is_named = std::integral_constant<bool, (predicate_builder::choose_uniqueness_tag<hd>::value >= 0)>

"Uniqueness Tag" is just the name of this predicate.

If this predicate doesn't have a name yet, it's just the name of the dependency-predicate; for example, if this is E(pred), then it would be pred's name. if no predicate has a name yet, this is -1.
